##### Terminal Ortamında PostgreSQL Sorguları #####

	//// Yaygın Kullanılan PSQL Terminal Komutları \\\\
- Postgresql'e baglanma islemleri
     > psql -U [username] -h localhost
     > psql -h localhost -p 5432 -U [username] [database]
     > psql -d [database] -U [username] -W
     > psql -h host -d [database] -U [username] -W
     > psql -U [username] -h host "dbname=db sslmode=require"
- \l: var olan databaseleri listeler
- \c [dbname]: var olan baska bir databasee baglanır
- \dt: databasede var olan tabloları listeler
- \d [tablename]: tablo hakkında bilgi verir
- \dv: databasede var olan view'leri listeler
- \df: databasede var olan fonksiyonları listeler
- \du: kullanıcıları ve atama rollerini listeler
- \g: son komutu tekrar eder
- \s: komut gecmisini görüntüler
- \s [filename]: komut gecmisini ilgili dosyaya kaydeder
- \i [filename]: bir dosyadan sql komutlarını import eder
- \?: bütün psql komutlarını tanımlar
- \h [name]: bir komut hakkında bilgi verir
     > örnek, \h ALTER TABLE
- \timing: sorgu süresini acar veya kapatır
- \a: output cıktısının seklini degistirir
- \e: son komutu düzenlemeye olanak saglar
- \H: output cıktısını HTML seklinde verir
- \x: okumayı kolaylastırır (kucuk outputlarda kullanılmalı)
- \q: psql shellden cıkar
- SELECT Version(): mevcut psql versiyonunu tanımlar
- \copy [query] TO 'path/[filename.csv]' DELIMITER ',' CSV HEADER;
	sorgunun sonucunu csv dosyasına kaydeder
